# Student Grade Manager (Python)

A simple console-based Student Grade Management System developed using Python.

This project allows users to manage student records, calculate grades, and view class performance statistics.

It was built as part of my programming practice during my Bachelor of Information and Communication Technology (Artificial Intelligence) at the University of Tasmania.

---

## 🚀 Features

- Add new students with name, ID, and marks
- View all stored student records
- Automatically calculate grades based on marks
- Display class summary (average, highest, lowest marks)
- Menu-driven console interface

---

## 🧠 Concepts Used

- Python functions
- Conditional statements
- Loops
- Lists and data structures
- Dataclasses
- Menu-driven program design

---

## 📊 Grading System

| Marks Range | Grade |
|------------|-------|
| 80 - 100   | HD    |
| 70 - 79    | DN    |
| 60 - 69    | CR    |
| 50 - 59    | PP    |
| Below 50   | NN    |
